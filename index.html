trt.TechnikJeopardy

Jeopardy-Webapp fuer den Technikunterricht (Bildungsplan BW 2016)

Ueber das Projekt

Dieses Projekt stellt eine Flask-basierte Jeopardy-Webapp mit Admin-Oberflaeche fuer den Technikunterricht bereit. Das Spiel orientiert sich an den Kompetenzbereichen des Bildungsplans Technik 2016 fuer Baden-Wuerttemberg und enthaelt einen Fragenpool mit 100 Fragen.

Zum GitHub-Repo

Installation & Betrieb

Die Anwendung kann lokal mit Python, ueber Docker Compose oder als Portainer-Stack betrieben werden:

git clone https://github.com/jbkunama1/trt.TechnikJeopardy.git

pip install -r requirements.txt && python app.py

oder

docker compose up -d

Vollstaendige Anleitung

Adminbereich & Fragenpool

Im Adminbereich (/admin) lassen sich Fragen nach Kategorie, Klassenstufe und Kompetenzbereich anlegen, bearbeiten und loeschen. Der mitgelieferte Fragenpool mit 100 Fragen kann laufend erweitert und an unterschiedliche Lerngruppen angepasst werden.

Sicherheit

Das Repository wird per GitHub Action automatisch mit TruffleHog auf versehentlich eingecheckte Secrets gescannt.
